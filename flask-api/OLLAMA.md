
# OLLAMA

## Prerequisites

1. **Ollama** [Descargar](https://ollama.com/download)

## Comands

1. **Download the llama3 model on your local**

   ```bash
   ollama run llama3:latest
   # install the one that has 8B not 70B params
   ```

2. **Too see the list of your local models**:
   ```bash
   ollama list
   ```
